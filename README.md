## Install ##
1. Copy *mvnc* script to directory `~/bin/`
2. Add execution permission to script : `chmod +x ~/bin/mvnc`
3. Add alias to shell ex: .bashrc : `alias mvn=~/bin/mvnc`
4. Reload .bashrc : `source ~/.bashrc`
